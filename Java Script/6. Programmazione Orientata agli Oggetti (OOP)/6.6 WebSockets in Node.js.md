## ðŸ”¹ 4. Broadcast: Inviare messaggi a tutti i client connessi

Possiamo inviare un messaggio a **tutti i client connessi**.

### âœ… **Modifica del server per inviare messaggi a tutti i client**

```js
server.on("connection", socket => {
    console.log("Nuovo client connesso!");

    socket.on("message", message => {
        console.log("Messaggio ricevuto:", message);

        // Invia il messaggio a tutti i client connessi
        server.clients.forEach(client => {
            if (client.readyState === WebSocket.OPEN) {
                client.send(message);
            }
        });
    });
});
```

ðŸ“Œ **Ora ogni client riceve i messaggi inviati da altri client.**

## ðŸ”¹ 6. Riconnessione automatica in caso di disconnessione

Se il client perde la connessione, possiamo riaprire il WebSocket automaticamente.

### âœ… **Client con riconnessione automatica**

```js
let socket;
function connettiWebSocket() {
    socket = new WebSocket("ws://localhost:8080");

    socket.addEventListener("open", () => console.log("Connesso!"));
    
    socket.addEventListener("close", () => {
        console.log("Connessione persa. Riconnessione...");
        setTimeout(connettiWebSocket, 3000);
    });
}

connettiWebSocket();
```

ðŸ“Œ **Se la connessione si chiude, il client prova a riconnettersi dopo 3 secondi.**
